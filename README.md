# telegrambot

Lightweight Spring Boot library for working with Telegram bots.

Supports:
- outbound messages (send to personal or group chat)
- optional inbound long-polling (can be disabled)

## Installation (JitPack)

```xml
<dependency>
  <groupId>com.github.NGorov</groupId>
  <artifactId>telegrambot</artifactId>
  <version>1.0.0</version>
</dependency>
